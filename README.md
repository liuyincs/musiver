## 🎼 功能概览

音流（Musiver）是一款**面向自有音乐库用户的音乐播放客户端**，核心目标是为 NAS / 私有音乐服务提供一个统一、稳定、体验友好的播放入口。

音流本身 **不提供任何音乐内容**，也不托管音频资源，仅作为播放与管理界面存在。

## ✨ 特点

- 📱 多平台原生体验
- ▶️ 完整的播放能力
- 🗂️ 基于元数据的音乐库浏览
- 🎨 封面、歌词与信息展示
- 🔄 多服务器与多线路支持

## 🔗 支持的媒体服务类型

音流支持连接多种常见的私有媒体服务，包括：

- Navidrome
- Emby
- Plex
- Jellyfin
- Subsonic / OpenSubsonic
- 群晖 Audio Station
- Audiobookshelf

不同服务在功能细节上可能存在差异，音流会在能力允许范围内提供统一的交互体验。

## 📱 平台支持

音流致力于提供多平台一致体验，目前已覆盖或规划支持：

- Android / Android TV
- iOS / macOS / tvOS
- HarmonyOS Next
- Windows
- Web

不同平台在 UI 表现上遵循各自平台的交互规范，但核心能力保持一致。
